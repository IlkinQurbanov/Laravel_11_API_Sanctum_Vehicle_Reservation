Документация API
Регистрация и Вход
POST /register
Регистрация нового пользователя. Необходимы поля: name, email, password, c_password, role.

POST /login
Вход пользователя. Необходимы поля: email, password.

Доступные автомобили
GET /available-vehicles
Получение списка доступных автомобилей для текущего пользователя. Доступно только для авторизованных пользователей.
Категории автомобилей
GET /categories
Получение списка всех категорий автомобилей. Доступно только для авторизованных пользователей.

GET /categories/{id}
Получение информации о категории по ID. Доступно только для авторизованных пользователей.

POST /categories
Создание новой категории автомобиля. Доступно только для администраторов.

PUT /categories/{id}
Обновление информации о категории автомобиля по ID. Доступно только для администраторов.

DELETE /categories/{id}
Удаление категории автомобиля по ID. Доступно только для администраторов.
